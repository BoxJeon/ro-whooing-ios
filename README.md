# ro-whooing-ios
- Read only 'whooing.com' iOS client.

## Phase 1
- login
- show balance sheet

## Support
- ios 8.0+
- xcode 6.3+

## Settings
```
[sudo] gem install cocoapods
pod setup
```

```
git clone https://github.com/BoxJeon/ro-whooing-ios.git
cd ro-whooing-ios
pod install
```

- open ro-whooing.xcworkspace
